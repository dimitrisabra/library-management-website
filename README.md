# Library Management System

## Description
This is a CRUD-based Library Management System built with PHP and MySQL. It is designed to efficiently manage books, users, and transactions in a library setting.

## Features
- **Manage Books**: Add, edit, delete, and view books in the library.
- **User Management**: Manage library users including registration, updates, and deleting user records.
- **Transaction Handling**: Keep track of borrowing and returning books.

## Technologies Used
- **PHP**: Server-side scripting language for backend.
- **MySQL**: Database management system used for data storage.

## Getting Started
1. Clone the repository: `git clone https://github.com/dimitrisabra/library-management-website`
2. Configure the database settings in the configuration file.
3. Run the application in a supported web server environment (e.g., XAMPP, LAMP).
